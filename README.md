# xlsxjs

This is a one-off, standalone fork of ExcelJS (MIT-licensed) maintained at `silverstone-i/xlsxjs` for project-specific needs.

## Status
- Not intended as a drop-in replacement for upstream.
- No guarantee of compatibility with ExcelJS releases.

## Install
This repository is not published to npm under `exceljs`.
If you want to consume it, use a Git dependency:

```sh
npm install git+ssh://git@github.com/silverstone-i/xlsxjs.git
```
